## 新增 endpoint（4 个）

- `POST /api/autopilot/dry-run` { event } → matched/actions/skipped
- `GET /api/rooms/:id/tasks/:tid/diff?from=N&to=M` → unified diff
- `GET /api/mcp/call-history?limit=100` → 最近 N 条 jsonl call log
- `POST /api/knowledge/:name/search?hybrid=1` → 走 hybridSearch

## UI 新增

- inspector 第 7 个 tab：🔬 Debate state 实时观察

- squad attempt drawer：📐 对比上次 attempt 按钮
- report 完成 toast：assertion 不通过项警告

## 风险与限制

1. **lobe-tokens-extension.css** 命名空间隔离，未启用零风险；启用后视觉对比未全验证
2. **hybridSearch** 需要 KB chunk 全部 embedded（ollama 在跑）才能真融合，否则降级 BM25
3. **debate_state_meta** broadcast 频次随 round 数量，长辩论会有很多事件
4. **call-logger** jsonl 月切，单月可能上 MB；未实现自动归档
