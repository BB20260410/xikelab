常见问题

Pro 是订阅还是买断？

买断。一次 $19 永久使用 + 所有未来更新。我们不喜欢订阅疲劳。

Free 和 Pro 的本质区别？

Free 让你试用 chat/debate 两种最基础模式；Pro 解锁完整 4 模式（含 squad 团队拆活和 arena 联网核对）+ 不限 MCP + 不限 adapter。

数据真的全本地么？

是的。panel 是 macOS 桌面应用，所有对话历史、配置、license 文件都存 ~/.claude-panel/，权限 0o600（仅你可读）。我们没有服务器后端，做不到收集你的数据。

license 怎么发？

购买后 LS/Polar 自动发邮件给你，里面是一串 license key。打开 panel → 点底部 Free 徽章 → 粘贴 → 激活。

能退款么？

30 天无理由退款。发邮件到 support@panel.app 即可。

支持哪些 AI？

Claude (Anthropic)、GPT (OpenAI)、Gemini (Google)、MiniMax、Ollama 本地模型、Claude Code Router、Claude Code CLI 等 8+ provider。
