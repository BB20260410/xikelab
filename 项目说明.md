# 03 — Claude 可视化面板

> 多 Claude Code 会话管理 GUI 面板，UI 参考 Codex，配色按 Anthropic 品牌规范。

## 快速开始

```bash
cd ~/Desktop/00_项目/05_Claude可视化面板
npm install
npm start              # 浏览器模式 → http://localhost:5173
# 或
npm run electron       # 原生 macOS app
```

## 主要特性

- **多 Claude 会话**：同时管理 N 个独立 cwd 的 claude session
- **持久化**：重启自动恢复（~/.claude-panel/data.json）
- **文件浏览器**：右侧 tab，点文件 inline 到 chat
- **多 GUI 窗口**：⤴ Terminal 按钮单开 / ⤴⤴ 全开 批量
- **暗色模式**：prefers-color-scheme 自动切
- **Codex 风格**：3 栏布局 + 圆形 avatar + 玻璃磨砂 + spacing/radius/shadow 系统继承
